## 开发流程
1. 立项。我们打算做一个ToDo应用
2. 产品大致规划，竞品分析，交互圈（产品），项目人员，开发周期。
3. 技术选型。flask mongo数据库 bootstrap。
4. 开始编码。先搭个架子。根据业务机构确定方法/类名/引用资源
5. 完成一个基础流程（展示列表）或先给假数据搭建静态页面
6. 完成add.update  delete
7. bootstrap美化
